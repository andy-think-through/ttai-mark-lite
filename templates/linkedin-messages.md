# LinkedIn Message Templates

## Week 1
Blank connection request (no message).

## Week 2 (if connected)
```
Hi [Name], thanks for connecting. I've dropped you an email about the [lead] I spotted in [town] -- thought it might be relevant for you. Let me know if useful. Andy
```

## Week 3 (if connected)
```
[Name], dropped you another email about the [lead] in [town]. Also mentioned how I've been finding these consistently for a few local firms, if that's of interest. Andy
```

## Week 4 (if connected)
```
[Name], last email from me today -- sent you the [lead] in [town] plus a bit about how I could set this up properly for your firm. Happy to chat if useful, no pressure either way. Andy
```

## Rules

- Only send to personal profiles (URLs containing `/in/`). Skip company pages (`/company/`).
- Week 1: Send regardless of connection status.
- Weeks 2-4: Only send if connected (LI Connected status).
- All tasks are for manual execution. Never automate LinkedIn.
