# Email 2: Value + Decision Maker

**Purpose:** Provide 1 lead with named contact, add PS about ongoing value.
**Word target:** ~75 words
**Leads required:** 1 (with named contact)

## Template

```
Subject: [Project Name] -- found the contact too

Hi [Name],

Another [Region] project that could be one to go after:

[Project Name], [Town] -- [why relevant, using package language].

The person to speak to is [Contact Name] ([Contact Role]) at [Contact Company].

Thought it was worth passing on.

Thanks,
Andy

PS I'm finding 10-20 of these a week across the area. Let me know if you'd like to keep getting them.
```
