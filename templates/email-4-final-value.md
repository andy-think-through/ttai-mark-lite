# Email 4: Final Value + Last Chance

**Purpose:** Last email -- provide 2 leads, soft CTA for a call.
**Word target:** ~80 words
**Leads required:** 2

## Template

```
Subject: Last one from me -- [Project Name], [Town]

Hi [Name],

I'll make this the last one.

A couple more that look like they'll need [trade] work:

1. [Project 1], [Town] -- [why relevant, using package language]
2. [Project 2], [Town] -- [why relevant, using package language]

I find these every week for firms like yours. If you'd like me to set this up properly for your firm, happy to have a quick chat.

Either way, hope they've been useful.

Thanks,
Andy
```

## Notes

- If only 1 lead available: change "A couple more" to "one more" and remove numbered list format.
- Log partial lead match in Outreach_Log Notes.
