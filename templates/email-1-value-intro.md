# Email 1: Value + Who I Am

**Purpose:** First touch -- provide 2 leads, establish Andy's role.
**Word target:** ~65 words
**Leads required:** 2

## Subject Line Rotation

Cycle through these three formats in strict A-B-C order across each batch:

- **Format A:** `Spotted this for you -- [Project Name], [Town]`
- **Format B:** `[Project Name], [Town] -- could be relevant for you`
- **Format C:** `New [trade] work near you -- [Project Name]`

## Template

```
Subject: [Use one of the three formats above]

Hi [Name],

I track upcoming construction projects across [Region] and pass the relevant ones to a few local trade firms. Spotted a couple that look like they'll need [trade] packages:

1. [Project 1], [Town] -- [why relevant, using package language]
2. [Project 2], [Town] -- [why relevant, using package language]

I'm not involved in these -- I just find them and pass them on.

Thought they might be of interest.

Thanks,
Andy
```
