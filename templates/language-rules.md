# Language Rules

All outreach copy must follow these language rules. Use package language -- describe leads the way a trade owner thinks about work.

## Always Use / Never Use

| Always Use | Never Use |
|---|---|
| Work, enquiries, jobs | Leads, pipeline, opportunities |
| The right person, the person to speak to | Decision makers |
| Finding you work | Outbound, business development |
| Costs a fraction of hiring someone | Cost-effective solution |
| Powered by AI (as cost reason) | AI-powered (as headline) |
| Your firm | Your business, your organisation |
| I'm tracking / I'm finding | I've been tracking / I've found |
| Andy (sign-off) | Andy, Think Through AI |
| Thought these might be of interest | Thought these leads might be of interest |
| Looks like it'll need [trade] packages | Could require significant [trade] |
| Early civils likely needed | Associated infrastructure works |
| One to go after | An opportunity to bid |

## Package Language Guidance

Describe leads the way a trade owner thinks about work -- in terms of packages they'd bid for:

- "looks like it'll need groundworks and drainage packages"
- "early civils likely needed"
- "roofing package on a new-build"

Avoid planning-speak:

- NOT "could require significant groundworks"
- NOT "development requiring associated infrastructure"
